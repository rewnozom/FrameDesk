# FrameDesk

FrameDesk is a desktop project management and file-explorer workspace for people who handle many local projects at once.
It was built to reduce context switching, file hunting, and the daily friction that happens when projects are spread across folders, tools, frameworks, and working contexts.

Instead of forcing users into a rigid system, FrameDesk is designed to adapt to real technical work:

- mixed stacks
- local folders
- quick launches
- custom organization
- highly personal visual setups

## Why FrameDesk exists

Day-to-day work usually needs more than one tool at the same time.

FrameDesk was built to bring those needs together in one desktop workspace:

- file browsing
- launchers
- bookmarks
- workspace organization
- project grouping
- framework-oriented workflows

When projects live across many folders and naming styles, they become harder to navigate, harder to maintain, and easier to forget.

FrameDesk helps turn that into a structure people can actually work with.

## What it helps solve

FrameDesk makes it easier to:

- organize projects into categories, subcategories, families, groups, and uncategorized areas
- open a project in its preferred tool
- launch PowerShell, CMD, Explorer, or VS Code from the project context
- scan folders and detect runnable projects automatically
- keep different work contexts separated through data sources and profiles
- manage projects and files inside the same workspace

## Key features

- Group-based workspace flow, including project groups in the tab/sidebar workflow
- Split explorer view for working with project structure and real files side by side
- Framework-oriented tools for scanning, detecting, and organizing local project collections
- Auto-detection for runnable Node, Rust, Python, and PowerShell-related projects
- Theme control with both global presets and per-section overrides
- Visual customization for header, sidebar, explorer, panels, buttons, popups, icons, and directories

## Built for flexible workflows

One interface does not fit every working style.

FrameDesk gives users UI control without losing structure, including:

- full theme presets
- per-section theme overrides
- separate visual control for header, sidebar, explorer, panels, status areas, popups, and buttons
- icon styling and icon provider choices
- directory and file icon customization

## In short

FrameDesk is built for developers and technical users who want a better way to organize, launch, browse, and manage many local projects from one place.
